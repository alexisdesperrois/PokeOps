# PokeOps #

Installer la totalité des fichiers.

Installer GO sur le site : `https://go.dev/doc/`

Ouvrir CMD, aller jusqu'au chemin de votre application PokeOps et exécuter la commande : 
``` go
go run main.go
```
Ensuite lancer votre navigateur et taper :
```
http://127.0.0.1:8080/
```
puis entrer
